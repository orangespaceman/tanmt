How to install the app locally.


## Requirements

* node (and npm)


## Installation

Once you have cloned this repo, run through the following steps:


### Installation - frontend

* Install frontend dependencies:

  ```
  npm install
  ```

* Watch for frontend file changes during development:

  ```
  npm run watch
  ```
